## Crew Cut Calculator

When splitting heist money with a crew, the host sets a percentage for each member. The tool calculates your exact take.

```
Your Take = Total Payout × (Your % / 100)

Example (Casino Heist, 4 players, host keeps 40%, others split 60%):
Total: $2,115,000 (cash target)
Host: $2,115,000 × 0.40 = $846,000
Each crew member: $2,115,000 × 0.20 = $423,000
```
